# File Bridge 🌉

File Bridge is a simple and fast file-sharing application that allows users to transfer files seamlessly between a PC and a mobile device over a local network.

## ✨ Features

- 📁 Transfer files between PC and phone
- ⚡ Fast local network file sharing
- 🔒 Secure device-to-device communication
- 📱 Mobile-friendly interface
- 💻 Cross-platform support
- 🚀 No cables required
- 🎯 Simple and intuitive user experience
